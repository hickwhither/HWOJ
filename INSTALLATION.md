## FRONTEND
```sh
# install
cd EPenguinOJ-frontend
npm i

# dev run
npm run dev
```

## BACKEND (uv recommended)
```sh
# install
cd backend
uv sync

# dev run
uv run fastapi dev
```

- Run with python (create venv if necessary)
```sh
# install
cd backend
pip install -r requirements.txt

# dev run
fastapi dev
```
